# Nome do Projeto
Simulação SIR

Constroi um grafico do numero de S, I e R em funcao do tempo. Calcula estimativas de RO e V.

## Uso

Uso para obter aproximacoes e uma visao geral das variacoes de cada estado(S,I,R)
